# 诗经楚辞数据使用指南

## 数据文件

已为你创建了以下数据文件：

- **诗经数据**: `src/data/shijing.json` (50篇精选)
- **楚辞数据**: `src/data/chuci.json` (20篇精选)

## 数据结构

### 诗经数据格式

```json
{
  "name": "诗经",
  "description": "中国古代诗歌开端，收录西周至春秋中期诗歌",
  "total": 50,
  "poems": [
    {
      "title": "关雎",
      "chapter": "国风",
      "section": "周南",
      "content": "关关雎鸠，在河之洲...",
      "keywords": ["关关", "雎鸠", "窈窕", "淑女", ...]
    }
  ]
}
```

### 楚辞数据格式

```json
{
  "name": "楚辞",
  "description": "战国时期楚国诗歌总集，以屈原作品为主",
  "total": 20,
  "poems": [
    {
      "title": "离骚",
      "author": "屈原",
      "section": "离骚",
      "content": "帝高阳之苗裔兮...",
      "keywords": ["帝高", "高阳", "嘉名", ...]
    }
  ]
}
```

## 使用方法

### 1. 基础导入

```typescript
import { poetryNaming } from '@/utils/poetryNaming';
```

### 2. 从诗经搜索名字

```typescript
// 搜索包含"君子"的诗句，返回5个名字
const names = poetryNaming.searchFromShijing('君子', 5);

// 结果示例：
// [
//   {
//     name: "君子",
//     source: "诗经",
//     quote: "窈窕淑女，君子好逑。",
//     origin: "《诗经·国风·周南·关雎》"
//   },
//   ...
// ]
```

### 3. 从楚辞搜索名字

```typescript
// 搜索包含"芳"的诗句，返回5个名字
const names = poetryNaming.searchFromChuci('芳', 5);

// 结果示例：
// [
//   {
//     name: "琼芳",
//     source: "楚辞",
//     quote: "瑶席兮玉瑱，盍将把兮琼芳。",
//     origin: "《楚辞·九歌·东皇太一》 屈原"
//   },
//   ...
// ]
```

### 4. 根据性别推荐（遵循传统：男楚辞，女诗经）

```typescript
// 女孩名字（从诗经）
const femaleNames = poetryNaming.searchByGender('female', '美', 10);

// 男孩名字（从楚辞）
const maleNames = poetryNaming.searchByGender('male', '云', 10);
```

### 5. 综合搜索

```typescript
// 从诗经和楚辞中搜索，各返回一半
const allNames = poetryNaming.searchAll('清', 20);
```

### 6. 获取统计信息

```typescript
const stats = poetryNaming.getStats();
console.log(stats);

// 输出：
// {
//   shijing: {
//     name: "诗经",
//     description: "中国古代诗歌开端...",
//     total: 50
//   },
//   chuci: {
//     name: "楚辞",
//     description: "战国时期楚国诗歌总集...",
//     total: 20
//   }
// }
```

## 在 Vue 组件中使用

```vue
<script setup lang="ts">
import { ref } from 'vue';
import { poetryNaming } from '@/utils/poetryNaming';
import type { NameResult } from '@/types/poetry';

const keyword = ref('');
const gender = ref<'male' | 'female'>('female');
const nameResults = ref<NameResult[]>([]);

const searchNames = () => {
  nameResults.value = poetryNaming.searchByGender(
    gender.value,
    keyword.value || undefined,
    20
  );
};
</script>

<template>
  <div>
    <el-radio-group v-model="gender">
      <el-radio label="female">女孩（诗经）</el-radio>
      <el-radio label="male">男孩（楚辞）</el-radio>
    </el-radio-group>

    <el-input
      v-model="keyword"
      placeholder="输入关键字（可选）"
      @keyup.enter="searchNames"
    />

    <el-button @click="searchNames">搜索</el-button>

    <div v-for="result in nameResults" :key="result.name + result.origin">
      <h3>{{ result.name }}</h3>
      <p>{{ result.quote }}</p>
      <small>{{ result.origin }}</small>
    </div>
  </div>
</template>
```

## 集成到 AI 提示词

你可以将诗经楚辞数据集成到 AI 起名提示词中：

```typescript
import { poetryNaming } from '@/utils/poetryNaming';

// 获取诗经楚辞示例作为参考
const examples = poetryNaming.searchAll('', 10);

const prompt = `
请根据以下信息生成名字：
- 性别：${gender}
- 八字：${bazi}
- 期望寓意：${meaning}

参考古典诗词：
${examples.map(e => `${e.name} - ${e.quote} (${e.origin})`).join('\n')}

要求：
1. 名字要有美好寓意
2. 符合五行八字
3. 借鉴诗经楚辞的美感
`;
```

## 扩展数据

如果你想添加更多诗词：

1. 编辑 `src/data/shijing.json` 或 `src/data/chuci.json`
2. 按照相同的格式添加新的诗篇
3. 确保更新 `total` 字段

## 数据来源说明

- 诗经：精选50篇经典篇目，包含《关雎》《桃夭》《蒹葭》等
- 楚辞：精选20篇经典篇目，以屈原作品为主

所有数据均包含：
- 完整诗文内容
- 适合起名的关键词提取
- 详细的出处信息

## 注意事项

1. **编码问题**：JSON文件使用UTF-8编码，确保正确显示中文
2. **性能优化**：如果数据量很大，考虑使用分页或虚拟滚动
3. **扩展性**：可以继续添加唐诗、宋词等其他数据源，使用相同的模式

---

**提示**：这些数据已经过精心整理，每首诗都提取了适合起名的关键词。你可以根据项目需求自由使用和扩展！
