# 宝宝起名工具 - 产品需求文档（PRD）

## 1. 产品概述

### 1.1 产品名称
诗韵取名阁（暂定）

### 1.2 产品定位
一个基于中国古典文化的智能宝宝起名工具，以诗经、楚辞等传统典籍为灵感来源，结合生辰八字理论，为新生儿提供富有文化底蕴的名字推荐。

### 1.3 产品愿景
传承中华传统文化，让每个名字都有诗意与寓意，帮助父母为孩子取一个既美好又有文化内涵的名字。

### 1.4 目标用户
- 准父母、新生儿父母
- 对中国传统文化有兴趣的家庭
- 追求名字文化内涵和寓意的用户
- 年龄段：25-40岁
- 地域：全国范围，特别是一二线城市

## 2. 核心功能需求

### 2.1 基础起名功能

#### 2.1.1 按姓氏起名
**功能描述：**
- 用户输入宝宝姓氏
- 系统调用AI生成符合该姓氏的名字
- 名字来源于诗经、楚辞、唐诗宋词等古典文献
- 每个名字附带详细出处和寓意说明

**交互流程：**
1. 用户在首页输入姓氏（1-2个汉字）
2. 点击"开始起名"按钮
3. 系统展示加载动画（水墨晕染效果）
4. 显示5-10个推荐名字，每个名字展示：
   - 完整姓名
   - 名字拼音
   - 出处典籍（如：《诗经·小雅·鹿鸣》）
   - 原文诗句
   - 寓意解释
   - 五行属性

**验证规则：**
- 姓氏必填，1-2个汉字
- 不允许输入特殊字符和数字

#### 2.1.2 按生辰八字起名
**功能描述：**
- 用户输入宝宝的出生年月日时
- 系统计算生辰八字和五行缺失
- 生成符合五行补缺的名字
- 结合诗词典故选择合适用字

**交互流程：**
1. 用户选择"按生辰八字起名"模式
2. 输入以下信息：
   - 宝宝姓氏
   - 出生日期（年-月-日）
   - 出生时辰（精确到时）
   - 性别（男/女）
3. 系统计算八字和五行信息
4. 展示八字分析结果：
   - 天干地支
   - 五行分布（金木水火土）
   - 喜用神
5. 推荐符合五行的名字列表（5-10个）

**验证规则：**
- 所有字段必填
- 日期格式验证
- 时辰范围：0-23点

### 2.2 名字详情展示

#### 2.2.1 名字卡片设计
每个推荐的名字以卡片形式展示，包含：

**基本信息区：**
- 完整姓名（大字展示）
- 拼音标注
- 五行属性标签

**文化内涵区：**
- 典籍来源（如：诗经·周南·关雎）
- 原文诗句（配有书法字体）
- 现代汉语释义
- 名字寓意解读
- 字义解析（每个字的含义）

**评分区（可选）：**
- 文化底蕴：★★★★★
- 音韵和谐：★★★★☆
- 五行匹配：★★★★★
- 综合评分：95分

### 2.3 名字收藏功能

**功能描述：**
用户可以收藏喜欢的名字，方便后续查看和对比

**交互流程：**
1. 每个名字卡片右上角有"收藏"图标（心形）
2. 点击收藏，图标变红，提示"已收藏"
3. 导航栏有"我的收藏"入口
4. 收藏页面展示所有收藏的名字
5. 支持取消收藏、批量删除

**数据存储：**
- 使用localStorage存储收藏数据
- 支持导出收藏列表（PDF/图片）

### 2.4 名字对比功能

**功能描述：**
用户可以选择2-3个名字进行对比，辅助决策

**交互流程：**
1. 收藏页面每个名字卡片有"加入对比"选项
2. 选择2-3个名字后，点击"开始对比"
3. 对比页面并排展示选中的名字
4. 对比维度：
   - 文化出处对比
   - 寓意对比
   - 五行属性对比
   - 音韵评分对比
   - 综合评分对比

### 2.5 筛选和排序

**筛选条件：**
- 按典籍来源（诗经/楚辞/唐诗/宋词等）
- 按五行属性（金/木/水/火/土）
- 按字数（单字名/双字名）
- 按性别倾向（偏男性/偏女性/中性）

**排序方式：**
- 综合评分从高到低
- 按文化底蕴排序
- 按音韵和谐度排序

## 3. 用户体验流程

### 3.1 主流程
```
用户访问首页
    ↓
选择起名模式（姓氏/八字）
    ↓
输入相关信息
    ↓
AI生成名字（3-5秒加载）
    ↓
浏览名字列表和详情
    ↓
筛选/排序调整
    ↓
收藏喜欢的名字
    ↓
对比名字
    ↓
确定最终名字
```

### 3.2 页面结构

**首页：**
- 顶部：Logo + 导航栏（首页/我的收藏/使用说明）
- 主体：水墨画背景 + 起名表单区域
- 底部：版权信息

**名字列表页：**
- 顶部：筛选和排序工具栏
- 主体：名字卡片列表（瀑布流或网格布局）
- 侧边栏：八字信息展示（八字模式）

**名字详情页（弹窗/抽屉）：**
- 完整的名字信息展示
- 诗句朗读功能（可选）
- 分享按钮

**收藏页：**
- 收藏的名字列表
- 对比功能入口
- 导出功能

**对比页：**
- 并排展示2-3个名字
- 对比表格
- 返回收藏页按钮

## 4. 视觉设计要求

### 4.1 设计风格
**水墨画风 - 简洁雅致**

### 4.2 色彩方案
**主色调：**
- 墨黑：#1a1a1a（主要文字）
- 浅墨：#4a4a4a（次要文字）
- 宣纸白：#f5f5f0（背景色）
- 淡墨灰：#e0e0e0（分割线、边框）

**辅助色：**
- 朱砂红：#c9302c（收藏、强调）
- 青瓷绿：#7fb685（五行木）
- 琉璃蓝：#5b8db8（五行水）
- 赤金：#b8936f（五行金）

### 4.3 字体
- 标题：思源宋体 / 方正仿宋
- 正文：苹方 / 思源黑体
- 诗句：楷体 / 隶书

### 4.4 视觉元素
- 水墨晕染背景
- 竹简卷轴装饰
- 印章元素（Logo、装饰）
- 毛笔笔触动效
- 水波纹涟漪效果

### 4.5 动效设计
- 页面加载：水墨晕染动画
- 卡片展示：从下至上淡入
- 收藏按钮：心跳动画
- 筛选展开：卷轴展开效果

## 5. 技术需求

### 5.1 前端技术栈
- 框架：Vue.js 3
- 构建工具：Vite
- UI组件库：Element Plus（自定义主题）
- 状态管理：Pinia
- 动画库：GreenSock (GSAP) / Lottie

### 5.2 AI接口需求
- 调用大语言模型API（OpenAI / 文心一言 / 通义千问等）
- Prompt设计要求：
  - 严格基于诗经、楚辞等古典文献
  - 返回完整的出处和寓意
  - 考虑五行属性（八字模式）
  - 音韵优美、寓意美好

## 9. 风险与应对

### 9.1 技术风险
**风险：AI API调用失败或超时**
- 应对：实现重试机制、降级方案（备用API）

**风险：前端性能问题**
- 应对：虚拟列表、懒加载、图片优化

### 9.2 内容风险
**风险：AI生成的名字不够准确**
- 应对：优化Prompt、人工审核、用户反馈机制

**风险：典籍出处错误**
- 应对：建立典籍数据库校验、专家审核

### 9.3 用户体验风险
**风险：用户期望过高**
- 应对：明确产品定位、合理引导用户预期
